# Bug Bounty Research: Web Vulnerability Discovery

## Overview
Conducted comprehensive security assessments on live websites including anamera.com, spizon.lk, and api.myntra.com. Identified and responsibly disclosed multiple vulnerabilities.

## Tools Used
- Burp Suite Professional
- OWASP ZAP
- Nmap
- Amass
- Wafw00f

## Vulnerabilities Found
- Reflected XSS (CVSS 7.1 - High)
- CSRF in Profile Update (CVSS 8.1 - High)
- Clickjacking (CVSS 5.4 - Medium)
- API Information Disclosure (CVSS 4.3 - Medium)
- HTTP Parameter Pollution (CVSS 5.3 - Medium)
- WAF Fingerprinting (CVSS 3.7 - Low)

## Methodology
1. Reconnaissance (Amass, Sublist3r, Nmap)
2. Scanning (OWASP ZAP, Burp Suite)
3. Manual Testing
4. Exploit Development
5. Responsible Disclosure
